Resource Profile: VA MHV PHR MB Lab Report

Official URL: https://department-of-veterans-affairs.github.io/mhv-fhir-phr-mapping/StructureDefinition/VA.MHV.PHR.MBlabReport Version: 0.3.7
Standards status: Trial-use Maturity Level: 2 Computable Name: MHVMBlabReport

A profile showing how Microbiology LabReportTO is mapped into a FHIR DiagnosticReport, Observation, and Specimen.

For API details see:

  • DiagnosticReport shows all of the DiagnosticReport. Including Microbiology, Pathology, and Chem-Hem.

Usage:

  • Examples for this Resource Profile: DiagnosticReport/ex-MHV-labReport-1, DiagnosticReport/ex-MHV-labReport-2 and DiagnosticReport/ex-MHV-labReport-3
