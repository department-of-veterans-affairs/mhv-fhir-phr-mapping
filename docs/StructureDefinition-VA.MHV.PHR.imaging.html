Resource Profile: VA MHV PHR Radiology

Official URL: https://johnmoehrke.github.io/MHV-PHR/StructureDefinition/VA.MHV.PHR.imaging Version: 0.1.16-beta
Standards status: Trial-use Maturity Level: 1 Computable Name: MHVimagingExam

A profile on the DocumentReference resource for MHV PHR exposing Radiology note (ImagingExamTO) using FHIR API.

For API details see:

  • DocumentReference shows all uses of DocumentReference. Including Notes, Discharge Summary, EKG, and Imaging.

Usage:

  • Examples for this Resource Profile: DocumentReference/ex-MHV-imaging-0

    • these never appeared in examples

      • reportId
