Resource Profile: VA MHV PHR Location

Official URL: https://department-of-veterans-affairs.github.io/mhv-fhir-phr-mapping/StructureDefinition/VA.MHV.PHR.location Version: 0.2.4
Standards status: Trial-use Maturity Level: 1 Computable Name: MHVlocation

A profile on the Location resource for MHV PHR exposing Location using FHIR API.

  • based on US-Core for Location
  • identifier
    • system = urn:oid:2.16.840.1.113883.4.349
  • name is the name as known
  • always a contained resource, so not exposed RESTfully

NOT USING Mapping to VIA HospitalLocationTO.

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
