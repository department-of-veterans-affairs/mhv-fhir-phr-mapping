Resource Profile: VA MHV PHR Organization

Official URL: https://johnmoehrke.github.io/MHV-PHR/StructureDefinition/VA.MHV.PHR.organization Version: 0.1.7-beta
Standards status: Trial-use Maturity Level: 0 Computable Name: MHVorganization

A profile on the Organization resource for MHV PHR exposing Organization using FHIR API.

  • based on US-Core for Organization
  • used for Lab Site. In FHIR R4, the Lab site needs to go into .performer which can’t take a Location. (TechnicalDebt)

Map to VDIF labSiteID

Usage:

  • Examples for this Resource Profile: Lab Site 989
